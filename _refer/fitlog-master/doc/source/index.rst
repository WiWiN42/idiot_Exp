fitlog 中文文档
==========================

fitlog是一款集成了自动版本管理和自动日志记录两种功能的 Python 包，由复旦大学计算机科学技术学院 `自然语言处理与深度学习组`_ 的
`fastNLP`_ 团队开发的。它可以帮助你在进行实验时方便地保存当前的代码、参数和结果。

.. image:: /figures/fitlog_chart.png


.. _自然语言处理与深度学习组: https://xpqiu.github.io/

.. _fastNLP: https://github.com/fastnlp/fastNLP
